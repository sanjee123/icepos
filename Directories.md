# Directory structure #

Where you must put your own files.


# Details #

IcePOS root dir:
|Folder|Description|
|:-----|:----------|
|  \bin\| base folder for IcePOS .jar file;|
|  \lib\| common libraries which can be used by all (mysql, xml, etc)|
|  \plugins\| folder with IcePOS plugins;|
|  \javapos\| javapos drivers directory;|
